# UI 契約: Open BookKeeping Razor Pages

**日期**: 2026-05-11  
**語言**: 所有 UI 文字、驗證、toast/alert、空白狀態使用繁體中文 zh-TW。

## 全站導覽

主要導覽必須至少包含:

- 首頁 `/`
- 明細 `/Transactions`
- 新增 `/Transactions/Create`
- 報表 `/Reports`
- 設定或更多 `/Settings`

在手機寬度導覽可收合，但主要流程必須可由鍵盤操作。

## Page: 首頁 `/`

**用途**: 顯示本月摘要、帳戶餘額、預算進度與最近交易。

**資料契約**

- 本月收入總額
- 本月支出總額
- 本月結餘
- 各帳戶目前餘額
- 各分類預算使用率、剩餘或超支金額
- 最近交易列表，排除軟刪除交易

**空白狀態**

- 無交易時顯示「本月尚無紀錄」與新增交易入口。

## Page: 交易明細 `/Transactions`

**用途**: 查詢、篩選、編輯與刪除交易。

**篩選欄位**

| 欄位 | 規則 |
|------|------|
| `StartDate` | 可空；若提供，含起始日。 |
| `EndDate` | 可空；若提供，含結束日且不得早於 `StartDate`。 |
| `CategoryId` | 可空；存在且未封存。 |
| `AccountId` | 可空；存在且未封存。 |
| `MinAmount` | 可空；若提供必須 >= 0。 |
| `MaxAmount` | 可空；若提供必須 >= `MinAmount`。 |
| `Keyword` | 可空；搜尋範圍至少包含備註。 |

**行為**

- 篩選條件為 AND。
- 列表預設依交易日期與更新時間倒序。
- 10,000 筆資料情境必須分頁。
- 刪除必須要求確認並送出 POST，不得以 GET 改變狀態。

## Page: 新增交易 `/Transactions/Create`

**用途**: 建立收入或支出交易。

**表單欄位**

| 欄位 | 必填 | 規則 |
|------|------|------|
| `TransactionDate` | 是 | 不晚於 Asia/Taipei 今日。 |
| `Type` | 是 | 收入或支出。 |
| `Amount` | 是 | > 0，最多 2 位小數。 |
| `CategoryId` | 是 | 類型需符合交易類型。 |
| `AccountId` | 是 | 有效帳戶。 |
| `Note` | 否 | 清理或拒絕注入風險文字。 |

**成功**

- 儲存後重新導向明細或回到新增頁，顯示成功訊息。
- 首頁摘要與明細列表 1 秒內反映變更。

**失敗**

- 保留使用者已輸入內容。
- 欄位層級顯示可修正錯誤。

## Page: 編輯交易 `/Transactions/Edit/{id}`

**用途**: 更新既有交易。

**行為**

- 欄位與新增交易相同。
- 儲存後建立遮罩異動摘要。
- 多分頁同時編輯採最後完成儲存為準。
- 若交易已軟刪除，顯示不可編輯狀態或導向明細。

## Page: 刪除交易 `/Transactions/Delete/{id}`

**用途**: 確認並軟刪除交易。

**行為**

- GET 顯示遮罩摘要與確認內容。
- POST 執行軟刪除。
- 刪除後交易不出現在一般明細與報表。
- 保存 `DeletedAtUtc` 與 `DeletionSummary`。

## Page: 分類管理 `/Categories`

**用途**: 管理收入與支出分類。

**欄位**

- 名稱
- 類型
- 圖示或辨識標記
- 顯示順序

**規則**

- 首次使用提供預設分類。
- 同類型 normalized name 唯一。
- 已被交易引用時不得刪除；提供封存或要求先移轉交易。

## Page: 帳戶管理 `/Accounts`

**用途**: 管理資金來源與查看餘額。

**欄位**

- 名稱
- 類型
- 圖示或辨識標記
- 初始餘額
- 固定幣別 TWD

**規則**

- normalized name 全域唯一。
- 目前餘額由初始餘額與未刪除交易計算。
- 已被交易引用的帳戶不得破壞歷史資料。

## Page: 報表 `/Reports`

**用途**: 查看指定月份收入、支出、結餘、分類佔比與趨勢。

**輸入**

- `Month`: `YYYY-MM`，以 Asia/Taipei 月份解讀。

**輸出**

- 總收入
- 總支出
- 結餘
- 支出分類金額與百分比
- 每日或每月趨勢資料

**空白狀態**

- 無交易顯示「本月尚無紀錄」與新增交易入口。

## Page: 預算 `/Budgets`

**用途**: 設定與追蹤每月支出分類預算。

**欄位**

- 支出分類
- 預算月份
- 預算金額

**提醒**

- 使用率 >= 80% 且 <= 100% 顯示接近上限提醒。
- 使用率 > 100% 顯示超出預算警告與超出金額。

## Page: CSV 匯入 `/Csv/Import`

**用途**: 上傳固定六欄 CSV 並批次建立交易。

**行為**

- 檔案必須 < 5 MB。
- 有效資料列 <= 10,000。
- 錯誤列跳過，成功列建立交易。
- 不存在的分類依交易類型自動建立，摘要中揭露。
- 結果顯示成功筆數、失敗筆數、新增分類與每列錯誤原因。

## Page: CSV 匯出 `/Csv/Export`

**用途**: 依日期範圍下載交易 CSV。

**輸入**

- `StartDate` 可空
- `EndDate` 可空

**行為**

- 未指定日期範圍時匯出全部未刪除交易。
- 匯出欄位與順序見 [csv-format.md](./csv-format.md)。
- 備註特殊字元與公式風險必須安全處理。

## 回饋與可及性

- 成功訊息可自動消失。
- 警告與錯誤訊息必須可手動關閉。
- 表單錯誤必須連結到欄位。
- 主要操作可用鍵盤完成。
- 320px 手機畫面與常見桌面寬度不得發生文字、控制項或圖表重疊。
